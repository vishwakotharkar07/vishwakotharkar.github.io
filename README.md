# 🚗 Four-Lane Car Dodge Game

A simple browser-based car dodge game built using **HTML, CSS, and JavaScript**. The goal is to avoid incoming traffic and survive as long as possible while your score increases over time.

---

## 🎮 Features

* 4-lane traffic system
* Smooth lane switching controls
* Increasing difficulty (speed increases over time)
* Score and high score tracking
* Different types of obstacle vehicles (sedan, SUV, truck, sports, van)
* Interactive UI with start/restart functionality
* Keyboard and mouse controls

---

## 🕹️ Controls

* **Arrow Left (←) / A** → Move Left
* **Arrow Right (→) / D** → Move Right
* **Mouse Click** → Click on a lane to move the car

---

## 🚀 How to Run

1. Download or clone the project
2. Open the file:

   ```
   four_lane_car_game.html
   ```
3. Run it in any modern browser (Chrome, Edge, Firefox)

No installation or setup required ✅

---

## 🧠 Game Logic

* The player controls a car that stays in one of four lanes.
* Obstacles spawn randomly and move downward.
* The player must avoid collisions.
* Score increases based on survival time.
* Game ends when a collision is detected.
